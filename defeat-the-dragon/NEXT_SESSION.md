# ドラゴンをたおせ RPG — 次セッション指示

## これは何
ブラウザで動く「テキスト進行・戦闘のみ」の王道ファンタジーRPG。1ファイルHTML、素のJS、フォントDotGothic16、24x24ドット絵。
ローグライク的ループ：拠点 → 冒険（一本道の連戦） → 倒れる/クリアで拠点に戻り、経験値・ゴールド・ドロップ・仲間をまとめて反映。

## 実装済み
- 戦闘コマンド：たたかう / メラ / やくそう / まほうのみず / ぼうぎょ ＋ 指輪スキル、会心の一撃（クリティカル）
- 敵9体：スライム→マッシュ→ゴブリン→ウルフ→おおこうもり→まどうし→がいこつ→ゴーレム→ドラゴン（まどうしは呪文、ゴーレムは高防御）
- はぐれメタル：低確率出現・高防御・逃走・経験値大（魔法が有効）
- 装備：武器(atk)/防具(def)/指輪(skill)。同じ装備のダブり入手で +1 強化。かじやでゴールド強化
- 使い魔：撃破でまれに仲間化 → 戦闘で育ち Lv5 で進化（金色＋改名）
- お店（道具屋・かじや）、ゴールド、道中イベント（宝箱/罠/分かれ道/謎の老人）
- 効果音（WebAudio、🔊トグル）
- セーブ：localStorage（配布版）。拠点復帰/購入/装備変更で自動保存

## 次にやること（最優先）
エリア追加：森 / 洞窟 / 城 など複数エリアで敵テーブルを変える。
- 拠点からエリア選択、または ドラゴン撃破で より深いエリアを解放
- エリアごとに 敵・ボス・ドロップ・背景色 を変える
- 既存の「一本道ループ」構造をそのまま活かす

## バックログ（あとで）
使い魔の進化分岐／さらなる成長、装備への属性付与、モンスター図鑑・コレクション、全体の難易度調整

## ビルド方法・制約
- ソース構成：
  - `build_spr.js` … 24x24スプライトを座標指定＋左右ミラーで生成し `spr_literal.txt` を出力
  - `gen.js` … CSS/HTML/JS を組み立て `rpg.html` を生成。配布版は `dragon-rpg.html`
  - 生成手順：`node build_spr.js` →（必要時のみ）→ `node gen.js`
- 変更後は必ず node で (1) JS構文チェック (2) スプライトが全て24x24か検証
- 文章はひらがな主体の日本語。装飾は最小。スマホ縦で1画面に収まるコンパクトさを維持
- セーブは localStorage 版（配布前提）。Claudeのアーティファクト内では保存されない点に注意
