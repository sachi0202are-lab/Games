# ストリートチャンピオン — 引き継ぎメモ

ブラウザ完結のテキスト進行・戦闘RPG。`defeat-the-dragon/` のエンジンを流用し、
テーマを「格闘ゲーム／大会制覇」に差し替えたもの。

## あそびかた / コンセプト
- しゅぎょう（連戦）で けいけんちを つみ、たいかいに でて、ぜんたいかいを せいはする。
- **セーブスロットは3つ**（タイトル「▶はじめる」→スロット画面。あき=はじめから、データあり=つづきから/けす(2回タップで確定)）。
- しゅじんこうは 3人から えらぶ（ストリートファイター/KOF風のモチーフを汲んだオリジナル）。**性能差あり**（CHARMOD: flat＋レベル成長補正）:
  - **ジン**（リュウ風・からてか）… バランス型(補正0)。Lv1: HP30/き8/こう8。スタート: ドリンク+2
  - **ダンディ**（ダッドリー風・重量級ボクサー）… HP・こうげき高め＋成長大、き少なめ。Lv1: HP38/き6/こう10。スタート: レザーグローブ
  - **ネオ**（エド風・サイコ少年）… き大量＋成長大でひっさつ連発、HP低め。Lv1: HP26/き14/こう8。スタート: はどうのまき

## ビルド / 確認
スクリプトと同じフォルダ（__dirname）入出力。手順は元ゲームと同じ:
```
cd street-champion
node build_spr.js   # 24x24スプライト生成 → spr_literal.txt（全14体, "ALL 24x24 OK"を確認）
node gen.js         # champion.html（配布版）と rpg.html（dev, gitignore対象）を生成
```
- 配布ファイル: `champion.html`（ルート index.html から リンク済み）
- セーブキー: `champ_save_v1_0` / `_1` / `_2`（3スロット）/ 音声: `champ_sound`（旧ゲームと混ざらない）
- 構文/動作チェック済み（headless スモークテストで 新規→キャラ選択→連戦→拠点→ショップ を通し確認）

## エンジン流用ポイント（仕組みは元のまま、データだけ差し替え）
- 戦闘: パンチ / 必殺技（き4, キャラ毎に名称変化）/ ドリンク（HP回復）/ ガード ＋ わざスキル ＋ クリーンヒット
- 拠点(ホームジム)→たいかい選択→一本道の連戦→倒れる/制覇で拠点へ、報酬まとめ反映
- 育成: グローブ(atk)/どうぎ(def)/わざ(skill)、ダブりで+1強化、ジムで強化購入
- セコンド（=元の使い魔）: まれに敗者が仲間化→育成→Lv5でかくせい(進化)
- 道中イベント、効果音、localStorageセーブ

## データ（gen.js 内）
- `CHARS` … 3キャラ（名前/スプライト/必殺技名/説明/スタート装備kit）★この版で追加
- `CHARMOD` … キャラ性能差（hp/mp/atk の flat と レベル成長 hpg/mpg/atkg）。`maxHpFor/maxMpFor/atkFor(lv,char)` に反映
- スロット: `SLOTS=3` / `curSlot` / `slotKey(i)` / `slotMeta(i)` / `renderSlots()`
- `MONSTERS` … ファイター8 + ボス3（streetking/undermaster/grandmaster）
- `AREAS` … たいかい3つ: ろじうらのストリート / ちかのとうぎじょう / せかいだいかい
- `WEAPONS`(グローブ5) `ARMORS`(どうぎ5) `RINGS`(わざ3) `SKILLS` `FAMILIARS`(セコンド9) `EVO`
- レア枠: 「でんせつの ファイター」（はぐれメタル枠、高経験値・すぐ逃げる）

## スプライト（build_spr.js / 24x24）
- 共通の `fighter()` でファイティングポーズの人型を生成、個別に上書き
- しゅじんこう: jin / dandy / neo
- ざこ: punk / karateka / muay / boxer / ninja(呪文=とびどうぐ) / yoga(呪文=かえん) / wrestler(高防御) / sumo(高防御)
- ボス: streetking / undermaster / grandmaster(ラスボス・あんこくオーラ)
